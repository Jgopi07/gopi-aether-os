# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/Jgopi07/gopi-aether-os.git
```

Move into the project folder:

```bash
cd gopi-aether-os
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

# 🚀 Build Project

```bash
npm run build
```
